Dynamicssheet

Getting Started

To start, simply create a new Dynamicssheet object:

var css = new Dynamicssheet();

Adding Styles

Once an instance is created, it will create its own style element to which you can add styles in one of three ways:

css.add(ruleSetObject) - multiple style rules at once (shown in example below)
css.add(ruleString, styleSetObj) - multiple styles at once
css.add(ruleString, propertyString, valueString) - one at a time

Removing Styles

You can remove styles in large batches or one-at-a-time:

css.remove(ruleArray) - remove multiple rules at once by passing an array of strings which match rules
css.remove(ruleString) - remove one full rule outright
css.remove(ruleString, propertyString) - remove a single style

Example

Obviously I should offer a few more examples one day, but for now, here's this page's css styling, completely with the Dynamicssheet.add() method.

css.add({
	'body': {
		'color': '#777',
		'font-family': 'Arial, sans-serif',
		'max-width': '900px',
		'margin': '2em auto'
	},
	'a': {
		'border-bottom': '1px solid #777',
		'text-decoration': 'none',
		'color': '#999'
	},
	'a:hover': {
		'border-bottom': '1px solid transparent',
		'opacity': '0.8',
		'-webkit-transition': 'all .25s',
		'-moz-transition': 'all .25s',
		'-ms-transition': 'all .25s',
		'-o-transition': 'all .25s',
		'transition': 'all .25s'
	},
	'pre': {
		'background-color': '#eee',
		'border-left': '2px solid #777',
		'margin-left': '2em',
		'padding': '1em'
	}
});
